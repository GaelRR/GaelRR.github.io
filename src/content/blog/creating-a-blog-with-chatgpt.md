For the past week I have been building this site with the help of ChatGPT.

I wanted to create a website to develop some skills and share some of my ideas.

Some of the things that I was interested in working on:

- Web development - Html, CSS, JavaScript
- AI - Learning how AI tools work and how they can be used
- Copywriting - Become better at writing
- Sharing my ideas and my work early on

Requirements:

- Free 💰 (I only paid for my domain name)
- Easy to mantain blog 📰
- Good user experience on both mobile and desktop 📱

This blog has been created while on vacation.
While I had time off work I focused on spending time with loved ones and taking care of my health
So I only worked on this project for a few hours, most of them arguing with ChatGPT.
Despite not having the time, it only tok a week to have a pretty nice responsive website with a blog.

Next steps for this blog:

- Improve blog: Make it easier to mantain and improve user experience.
- Write more blogposts
